Navigation model

Sidebars: BARBER_NAV, BEAUTY_NAV, SPA_NAV, NAIL_BAR_NAV, CLINIC_NAV, MOBILE_NAV, THERAPY_NAV, PRODUCTS_NAV — each item: path, label, section, roles[], optional requiredFeature.

Multi-mode: prototype can use a BARBER_NAV superset when categories.length > 1. Production: merged manifest keyed by { mode, role }.

Branch / reception / staff

Branch: schedule, attendance, bookings, waitlist/queue (mode-dependent). Reception: live board, check-in, queue, schedule, QR clock, POS if plan. Staff: “My” workspace (chair/station/room/road/sessions) + earnings, clients, POS.
