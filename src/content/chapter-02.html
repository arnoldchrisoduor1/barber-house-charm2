Process topology

Clients use JSON over HTTPS; M-Pesa and provider callbacks hit Laravel with validation. Next.js is a consumer of the v1 API; Node handles calling.

Critical path — booking create

Next → POST bookings → Laravel auth/scope → BookingService → Postgres transaction + availability → Horizon (SMS) → Soketi broadcast → 201.

M-Pesa: STK POST → Daraja → callback → HMAC + IP + Redis idempotency → update transactions → broadcast POS.

Chat: GET/POST messages → broadcast private-org.{id}.chat.{channel} → Echo + TanStack patch.

Haus Connect: Next → Laravel session token → Node → provider → webhooks → call_logs.
